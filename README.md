# muddy

A MUD client in your browser usin' node.js & websockets.

## Installation & Usage

### Installation

    $ git clone git://github.com/dannytatom/muddy.git
    $ cd muddy
    $ npm install
    $ vim config/config.json
    $ node init.js &
    $ open http://localhost:6660
